# TON Stories Roadmap

This public roadmap describes the product direction at a high level only.

## Current Focus

- clarify the Telegram-first advertiser journey
- describe Telegram Stories as a performance advertising format
- support simple campaign request and status communication
- improve public discovery through SEO-friendly project documentation

## Product Direction

- advertiser onboarding through Telegram
- creator and channel placement discovery
- guided campaign preparation
- story publication coordination
- reporting concepts for story views and campaign outcomes

## Not Included Publicly

- production source code
- deployment instructions
- internal automation logic
- private data models
- pricing internals
- credentials or API configuration
